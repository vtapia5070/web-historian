#web-historian

Users can archive and retrieve website urls.
This is an application that demonstrates RESTful a client/server model using npm's http module.

This is a solo project I completed as a student at [Hack Reactor Remote Beta](http://www.hackreactor.com/remote-beta).

##Development
##Installing Dependencies
From within the root directory install all dependencies with the following command:

```
npm install
```
To start npm server run the following command:
```
npm start
```
Run Chai/Mocha test with the following command:
```
npm test
```
